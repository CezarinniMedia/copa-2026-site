## Story

**As a** proposal viewer (Ana Bruno, Ger. Marketing do Shopping Norte Sul Plaza),
**I want** to see visually striking sections that immediately communicate the Arena da Figurinha concept, its benefits, and the zero-cost partnership model,
**so that** I understand the value proposition within seconds and feel confident presenting this to the shopping's board of directors.

## Acceptance Criteria

1. **Hero Section:** "Arena da Figurinha | Copa do Mundo 2026" headline + "Josimar Producoes — Distribuidor Oficial Panini" subline. Gradient/pattern background (no photos). Strong visual impact. CTA button "Fale Conosco" to scroll to `#contato`
2. **O Conceito Section:** Describes the 170m2 themed space — campo de grama sintetica, mesas circulares, loja oficial Panini, espacos instagramaveis. Clean descriptive layout with Lucide icon accents
3. **Beneficios Section:** 8 benefit cards in responsive grid. Each card with Lucide icon + title + short description. Benefits from briefing:
   - Fluxo de pessoas (foot traffic)
   - Posicionamento unico (unique positioning in MS)
   - Engajamento familiar (family engagement)
   - Conteudo para redes (social media content)
   - Consignacao para lojistas (retailer consignment)
   - Zero custo (zero cost to shopping)
   - Midia digital (digital media campaign)
   - Experiencia tematica (themed experience)
4. **Modelo de Parceria Section:** STRONG visual highlight that this is **ZERO COST** for the shopping. This is the #1 selling point — must be impossible to miss. Use Copa accent colors, large typography, visual emphasis (e.g., large "R$ 0" or "CUSTO ZERO" with gradient background)
5. All text content extracted from briefing document — no invented content
6. All sections use `SectionWrapper` and `SectionHeading` from Story 1.1
7. Responsive: looks great on desktop (1280px+) and mobile (375px+)
8. Lucide React icons only (no emojis, no image files)
9. `npm run lint` passes
10. `npm run build` succeeds
