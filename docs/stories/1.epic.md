# Epic 1: Arena da Figurinha — Landing Page Implementation

## Epic Goal

Deliver a production-ready, visually striking proposal landing page that positions the Arena da Figurinha (Copa do Mundo 2026) as a premium partnership opportunity for Shopping Norte Sul Plaza, Campo Grande-MS. The page must communicate professionalism, credibility, and the zero-cost partnership model to Ana Bruno (Ger. Marketing) and diretoria.

- **Current state:** Next.js 15 scaffolding with boilerplate Create Next App content
- **Technology stack:** Next.js 15 (App Router), TypeScript, Tailwind CSS v4, shadcn/ui (button, card, badge, separator, table, accordion, tabs), Lucide React
- **Integration points:** Standalone site — no external APIs, database, or auth required

### Enhancement Details

- **What's being added:** Complete 10-section landing page replacing boilerplate, with custom Copa/Brasil design system
- **How it integrates:** Replaces `src/app/page.tsx` and extends `globals.css` with custom design tokens
- **Success criteria:**
  - All 10 sections implemented with real content from briefing
  - Professional dark-mode design with Brasil/Copa color scheme
  - Responsive (mobile-first, desktop-optimized for Ana's desktop)
  - Zero placeholder content — production-ready for client presentation
  - Lighthouse performance score >= 90

| Aspect | Specification |
|--------|--------------|
| **Typography** | Inter — Bold (700) for headlines, Regular (400) for body text. Via `next/font/google` |
| **Primary Colors** | Verde #009B3A, Amarelo #FFDF00, Azul #002776 |
| **Theme** | Dark mode (fundo escuro) with vibrant accent colors |
| **Icons** | Lucide React only. NEVER emojis |
